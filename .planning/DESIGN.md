# Дизайн-концепция: Oko VPN

Прототип по ТЗ допускает «дизайн не важен», но сильная подача — наш дифференциатор. Концепция рассчитана на реализацию за 3–5 часов внутри 48-часового бюджета: один экран, одна сигнатурная анимация, готовая тёмная и светлая темы.

## Направление

**Dark moody + утилитарная плотность данных.** VPN — про доверие и контроль: глубокий тёмный фон, точная типографика, один светящийся акцент, никакой декоративности. Атмосфера меняется вместе с состоянием туннеля: фоновое свечение и акцентный цвет ведёт `VpnState`.

Запрещено: фиолетово-синие градиенты, дефолтные Material-виджеты без стилизации, одинаковые карточки с одинаковыми отступами, спиннеры.

## Сигнатурный элемент: «Око»

Название проекта задаёт метафору. Центральный элемент экрана — кольцо-ирис (кастомный `CustomPainter` + `AnimationController`):

| Состояние | Поведение ириса | Цвет |
|---|---|---|
| Disconnected | Кольцо тонкое, статичное, «зрачок» закрыт | графитовый `#5A6472` |
| Connecting | Кольцо «дышит» (scale 0.97→1.03), сегмент бежит по окружности | янтарный `#FFB454` |
| Connected | Ирис раскрыт, мягкое свечение (радиальный glow), внутри таймер | изумрудно-бирюзовый `#2CE5A7` |
| Disconnecting | Свечение гаснет, сегменты схлопываются | янтарный `#FFB454` |
| Error | Кольцо разомкнуто, короткая вибрация shake + haptic | коралловый `#FF5C5C` |

Ирис — одновременно и индикатор, и «hero moment» для видео-демо: Connect → зрачок раскрывается → появляется таймер.

## Палитра

Тёмная тема (основная):

- Фон: `#0B0F14` (void, чуть тёплый чёрный) + радиальный glow от центра, тонированный цветом состояния (opacity 0.10–0.16)
- Поверхности: `#121821` (карточки), `#1A2230` (elevated)
- Текст: `#F2F5F9` primary, `#8B95A5` secondary
- Акценты состояний: `#2CE5A7` (connected), `#FFB454` (connecting/disconnecting), `#FF5C5C` (error), `#5A6472` (idle)

Светлая тема: фон `#F4F6F9`, поверхности белые, те же акценты с затемнением до контраста 4.5:1. Оба режима обязательны, тема через `ThemeExtension` (OkoTones), не хардкод в виджетах.

## Типографика

- Display / таймер / цифры трафика: **Space Grotesk** (tabular figures, крупный кегль таймера 40–48)
- Body / подписи: **Inter**
- Логи: **JetBrains Mono** 12sp, цветовая маркировка уровня (info — secondary, warning — янтарный, error — коралловый)

Пакет `google_fonts`, минимум кеглей: Display 48 / Title 20 / Body 15 / Caption 12 / Mono 12.

## Композиция экрана (портрет, phone-only)

Сверху вниз, с нарушением монотонной вертикали за счёт разных масштабов блоков:

1. **AppBar-зона** (hard zone): логотип-слово `oko`, справа — бейдж статуса словом (Disconnected/Connected…)
2. **Ирис** (стретч-зона, ~40% высоты): кольцо состояния; внутри — таймер подключения (`00:00:00`) в Connected, иконка закрытого глаза в Disconnected
3. **Карточка сервера**: флаг/имя сервера, адрес:порт из VpnConfig, шеврон (в прототипе — статичный выбранный сервер из VLESS-ссылки)
4. **Ряд трафика**: две компактные плитки `↓ down` / `↑ up` (данные из trafficChanged), tabular-цифры, единицы авто (KB/MB)
5. **Кнопка Connect / Disconnect** (thumb zone): полная ширина, высота 56, цвет ведёт состояние; в переходных состояниях — disabled с progress-сегментом; haptic `mediumImpact` на нажатие
6. **Панель логов**: `DraggableScrollableSheet` снизу (min 12% — видна последняя строка, max 70%); grabber, заголовок `Logs`, кнопка copy-all; строки моноширинные, автоскролл к новым событиям

Safe areas обязательны; touch targets ≥48dp; отступы по сетке 8.

## Motion

- Вход на экран: staggered fade+slide (ирис → карточки → кнопка), 350ms, `Curves.easeOutCubic`
- Смена состояния: цвет и glow интерполируются 300ms; переходы статуса никогда не «прыгают»
- Connected: единожды — короткий «раскрыв» ириса с лёгким overshoot (`easeOutBack`), затем статика + медленное дыхание glow (4s цикл)
- Error: shake 250ms + `HapticFeedback.heavyImpact`
- Уважать `MediaQuery.disableAnimations` — статичные состояния без дыхания

## Платформенная адаптация

- База — Material 3 (`ColorScheme` из палитры выше, не fromSeed), Android: edge-to-edge, NavigationBar не нужен (один экран)
- iOS: `CupertinoPageTransition`-поведение по умолчанию, swipe-back не переопределять, шрифтовая вёрстка выдерживает Dynamic Type до 200%
- Иконки: `lucide_icons` или Material Symbols rounded — единый набор

## Доступность

- Semantics-лейблы на ирисе («VPN подключён, 12 минут»), кнопке и плитках трафика
- Статус дублируется словом в бейдже — не только цветом
- Контраст всех пар проверен ≥4.5:1 (текст) / ≥3:1 (крупные элементы)

## Границы реализации

- Один экран + панель логов; никаких табов, онбординга, настроек
- Кастомная графика — только ирис (один CustomPainter); остальное — стилизованные стандартные виджеты
- Ассеты: без изображений, только шрифты и векторные иконки — репозиторий остаётся лёгким
