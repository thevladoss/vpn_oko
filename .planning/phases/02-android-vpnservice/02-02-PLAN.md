<objective>
Подготовить платформенную обвязку foreground-VPN-сервиса: манифест (permissions + декларация сервиса с BIND_VPN_SERVICE и `foregroundServiceType=systemExempted`) и фабрику уведомления (канал API 26 + ongoing-нотификация).
