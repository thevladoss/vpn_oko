    Запустить на эмуляторе/устройстве Android API 34+ (`flutter run`), выполнить чеклист 03-VALIDATION ## Manual-Only Verifications:
    1. Пять статусов (UI-01, UI-06): пройти Disconnect → Connect → Connecting → Connected → Disconnect; ирис на каждом статусе анимируется и красится верно (idle графит, connecting amber+бегущий сегмент+дыхание, connected green+раскрыв зрачка+glow+таймер, disconnecting гаснет, error coral+shake+haptic).     4. Логи (UI-03, UI-08): строки идут вживую, уровни info/warning/error окрашены, прокрутка вверх ставит автоскролл на паузу, кнопка copy-all копирует в буфер (SnackBar 'Copied N lines').
    5. Таймер + сервер + трафик (UI-04, UI-05): в Connected таймер тикает от connectedSince (переключить тему/скролл логов — таймер НЕ сбрасывается); карточка сервера показывает name + host:port; плитки rx/tx растут при пинге в подсеть туннеля.

    7. Reduce-motion: включить «Убрать анимацию» в системе — ирис рисует статичные кадры без дыхания/сегмента.
