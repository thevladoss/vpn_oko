## Checkpoint 1 — публичный репозиторий + push (шаг пользователя)
git remote не настроен (решение «пока локально»). При сдаче:
1. Создать публичный репозиторий GitHub/GitLab.

4. Дождаться первого зелёного CI-прогона на GitHub Actions → бейдж активен (DOC-04).
