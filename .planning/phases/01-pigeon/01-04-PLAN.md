VpnRepositoryImpl(VpnNativeDatasource ds) implements VpnRepository:
- в конструкторе подписывается на ds.states → кэширует _last, пушит в broadcast _controller
- watchState(): async* yield _last; yield* _controller.stream   (replay последнего первым)
- watchTraffic(): проксирует ds.traffic
- connect(VpnConfig) → маппит VpnConfig→VpnConfigMessage, ds.start(...); ловит PlatformException → typed Failure (throw/через результат)
- disconnect() → ds.stop()
- syncStatus(): _last = statusSnapshotToEntity(await ds.currentStatus()); _controller.add(_last)
- _last стартует как VpnDisconnected()

LogRepositoryImpl(LogNativeDatasource ds) implements LogRepository:
- ring buffer (лимит ~500) + broadcast; watchLogs(): yield буфер, затем поток

AppDependencies (app/di.dart):
- _bridge = VpnBridge(hostApi: VpnHostApi(), events: vpnEvents())   // реальный стрим здесь
- ds = VpnNativeDatasource(_bridge); logDs = LogNativeDatasource(_bridge)
- vpnRepository = VpnRepositoryImpl(ds); logRepository = LogRepositoryImpl(logDs)
- usecases: watchVpnState, connectVpn, disconnectVpn, syncStatus, watchLogs

vpn_exception.dart: mapPlatformException(PlatformException) → Failure (PlatformFailure/VpnStartFailure)
</interfaces>
