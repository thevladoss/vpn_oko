## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Kotlin эмиттер → Flutter engine (event channel) | события пересекают границу процесс-платформа → Dart; требуется main thread |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-1-02 | Denial of Service | VpnEventListener / VpnEventBus.emit | mitigate | Каждый sink.success обёрнут в Handler(Looper.getMainLooper()).post; прямой вызов sink из другого потока запрещён (Pitfall 1). emit допускается из любого потока — доставка централизована на main |
| T-1-06 | Spoofing (гонка статуса) | VpnEventBus.addListener | mitigate | onListen реплеит lastStatus подписчику сразу; getStatus() отдаёт snapshot из кэша (Pitfall 2, BRG-04) |
| T-1-08 | Tampering (контракт level) | echo LogMessage.level | mitigate | level ставится строковым литералом нижнего регистра "info", совпадающим с именами Dart-enum LogLevel; верхний/смешанный регистр запрещён, иначе byName в маппере 01-03 бросит ArgumentError на live-прогоне (Plan 07) |
| T-1-01 | Information Disclosure | echo LogMessage-тексты | mitigate | Тексты синтетические ("starting"/"connected"); UUID/поля VpnConfig в лог не кладём |
| T-1-EP | Elevation of Privilege | новые Android-компоненты | accept | Экспортируемых компонентов и VPN-permissions в Phase 1 не добавляем; BIND_VPN_SERVICE/FGS — Phase 2 |
</threat_model>
